# DataCentreIQ

Professional hackathon prototype for an AI-powered EPC Project Intelligence platform for data centre construction.

## Run

```bash
npm.cmd run start
```

Open:

```text
http://localhost:4173
```

The app is dependency-free and can also run with:

```bash
node server.mjs
```

## Prototype Coverage

- Command centre with live risk propagation visualisation.
- Specification and quality compliance agent with cited deviation evidence.
- Predictive schedule risk engine with mitigation scenario generation.
- Supply chain route monitor with critical equipment alerts.
- Commissioning QA copilot with IST sequence tracking and auto-record IDs.
- RFI intelligence chat with similar resolved precedent.
- Architecture view suitable for a presentation deck slide.

## Judging Mapping

- Innovation: multi-agent project twin for EPC delivery.
- Business impact: prevents procurement rework, schedule slips, and commissioning retests.
- Technical excellence: modular local data model, responsive UI, no external dependency risk.
- Scalability: separates ingestion, project graph, agents, workflows, and QMS audit trail.
- User experience: operational dashboard first, with direct review and mitigation actions.
